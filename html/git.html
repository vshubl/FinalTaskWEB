Git

  • git clone - скопировать удалённый репозиторий на компьютер.
  • git push - внесение изменений на удалённый репозиторий.
  • git log - вывод списка коммитов.
  • git pull - перенос изменений из удалённого репозитория в локальный.
  • git branch - создание ветки в репозитории.
